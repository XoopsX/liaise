Information
----------------------
Current Version: 1.26 / 1.5
Liaise is a module for XOOPS offers widely customizable contact forms. Liaise generates different kind of form elements e.g. text areas, check boxes for webmasters to create their "Contact Us" forms by desire. Submitted information can be sent by email or through the private message system of XOOPS, and received by selected user group.
Key features:
 - Unlimited contact forms for different groups users and receivers.
 - Customizable form elements include text boxes, text areaa, selection boxes, radio buttons and check boxes.
 - Toggle the visibility and display order of the forms and elements on the fly.
 - Information submits through email or private message.
 - Submitted information received by a specific group of users.
 - Use of Smarty templates.

System Requirements
----------------------
Liaise 1.26: XOOPS 2.0.12-13.2 ; XOOPS Cube 2.0.12/13a
Liaise 1.5: XOOPS 2.2.3a
PHP 4.3.x or later

Using Liaise
----------------------
To install
To install, go install it just like installing any module of XOOPS. You know you are smart enough to make it right.
If you want an installation in a specific language other than in English, please use the files under "extras/your-language" folder and follow the instructions detailed on the readme file (if available).
For more information of installing a module of XOOPS, please refer to:
http://www.xoops.org/modules/smartfaq/faq.php?faqid=90

To use
Lists of form elements are provided for the contact us form by default. You can edit, delete, clone or reorder the form elements to match your need; go to the module preferences for editing various configurations. You can be handy with it in less than five minutes.
You can also edit the Smarty template of the form to match the look and feel of your site.
For detailed documentation please the Liaise home page:
http://www.brandycoke.com/products/liaise/
